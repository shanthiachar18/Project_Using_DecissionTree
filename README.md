# Objective
To build a predictive model for IPhone purchase.

# Overview

Performed EDA to get more informative features like null values, outliers, correlation and so on.

# 2.Model Building:
Used DecisionTreeClassifier model.

# 3. Model Validation:

 * Confusion matrix
